<h1 align="center">JetMovieApp</h1>

<p align="center">
JetMovieApp is a sample Android project using Firestore service database

## Tools and Techs
* Language Kotlin
* Pattern architecture MVVM
* Jetpack Compose to declarative UI
* Coroutines + Flow
* Firebase Firestore
* Dagger Hilt to Dependency Injection
* Coil to async image
* Splash Screen API
* DataStore
